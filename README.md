# Shopaholic

Simple Next.js demo project with a shopping list UI.

Getting started

1. Install dependencies

```
npm install
```

2. Run dev server

```
npm run dev
```

3. Build for production

```
npm run build
npm start
```

- The project uses the app/ directory with client components.
